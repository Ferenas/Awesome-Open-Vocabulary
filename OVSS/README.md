# Open Vocabulary Semantic Segmentation

## Keywords

- `cap.`: Use caption as training data
- `vlm.`: Use pretrained VLMs like CLIP 
- `pl.`: Generate pseudo labels
- `w/o ps.`: Without pixel-level supervision
